# Montessori Academy LMS

Employee Learning Management System  
Montessori & Friends Pflegedienst GmbH, Berlin

## Overview
A fully browser-based LMS built for care staff. No installation required — runs in any modern browser.

## Features
- Employee login with progress tracking per staff member
- 27 medical and nursing courses (English / German bilingual)
- Each course has 3 modules and a 10-question exam (75% pass mark)
- Admin panel with real-time Employee Tracker
- Reports by course and by employee with CSV export
- Works fully offline after loading

## Access
Open the LMS in any browser using the deployed link.

Employee Login: Enter Full Name, Employee ID (e.g. MF-001), and Department.  
Admin Login: Click Admin Access — contact IT for the password.

## Tech Stack
Pure HTML, CSS, JavaScript. No frameworks or dependencies.  
Progress saved in browser localStorage.
